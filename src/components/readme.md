# 组件参数

## 头像

### 属性
| 参数  | 说明 | 类型 | 可选值 | 默认值 |
| :---- | :--- | :---| :-----| :------|
| src   | 图片地址| string | —— | ——   |
| alt   | 图片地址错误时显示的文字 | string | —— | 这是一张占位头像  |
| size   | 图片大小 | number | —— | 100 |

## 图标
 
### 属性
| 参数  | 说明 | 类型 | 可选值 | 默认值 |
| :---- | :--- | :---| :-----| :------|
| type   | 图标类名 | string | —— | home  |
